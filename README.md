# python-playground-tasks
python-playground-tasks is a repository containing several seperate small projects, namely:
- anagram_checker: Function that checks if 2 strings are anagrams
- string_unique_chars: Function that checks if all chars in a string are unique
- current_time: Function that gets the current time
- subtract_five_days: Function that subtracts five days from the current day
- missing_array_numbers: Function that find the missing number in a given array of numbers between 10 and 20
- fibonacci: Method that gets the first 10 Fibonacci numbers and prints them
- factorial: Functions that find the factorial recursively and iteratively

## Installation
### Requirements
- Python 3.11 or higher

### Install locally
Clone the repository and install the package.

## Usage
First, import the function you require, then call it.

## Testing
This project uses unittest and pytest for testing.
